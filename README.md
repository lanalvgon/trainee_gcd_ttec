## Sobre o Projeto

Desenvolvimento do projeto de comunicação de TT\&C do Projeto Trainee 2025 da Gama Cube Design. Os códigos foram testados utilizando o microcontrolador ESP32 e módulos LoRa RA-01 e RA-02.
